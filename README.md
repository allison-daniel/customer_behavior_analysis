# Customer Behavior Analysis

## Project Overview

This project analyzes customer shopping behavior data using Python, PostgreSQL, and Power BI. The objective is to identify customer purchasing patterns, product performance, revenue trends, and customer segments through data cleaning, exploratory data analysis, SQL-based business insights, and interactive dashboards.

## Dataset

* 3,900 customer purchase records
* 18 original features including demographics, product information, purchasing behavior, shipping preferences, subscription status, and review ratings

## Tools & Technologies

* Python
* Pandas
* PostgreSQL
* SQL
* SQLAlchemy
* Power BI
* Jupyter Notebook

## Data Preprocessing & Feature Engineering

The dataset was cleaned and transformed using Python:

* Handled missing values in `review_rating` using category-wise median imputation
* Standardized column names
* Created customer age segments:

  * Young Adult
  * Adult
  * Middle Aged
  * Senior
* Converted purchase frequency categories into numerical day values
* Identified and removed redundant columns (`promo_code_used`)
* Loaded the cleaned dataset into PostgreSQL using SQLAlchemy

## SQL Business Questions Answered

1. Total revenue generated by male vs female customers
2. Customers who applied discounts but spent above average
3. Top 5 products with highest average review ratings
4. Comparison of average purchase amounts between shipping methods
5. Spending behavior of subscribed vs non-subscribed customers
6. Products with the highest discount utilization rates
7. Customer segmentation into New, Returning, and Loyal customers
8. Top 3 most purchased products within each category
9. Relationship between repeat purchases and subscription status
10. Revenue contribution by customer age groups

## Dashboard Insights

The Power BI dashboard provides visual analysis of:

* Revenue by product category
* Revenue by age group
* Sales by product category
* Sales by age group
* Average purchase amouunt
* Customer vs. Subscription behavior

## Project Workflow
